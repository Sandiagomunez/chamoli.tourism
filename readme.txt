Chamoli Tourism Website - Local Setup Guide

1. Requirements:
   - Node.js (v16+)
   - npm (v8+)

2. Steps to run:
   a. Extract this zip file.
   b. Open terminal in the root folder.
   c. Run: npm install
   d. Run: npm run dev
   e. Visit: http://localhost:5173

3. Offline Build:
   a. Run: npm run build
   b. Open /dist/index.html in browser

This site includes:
- ReactJS + Vite + TailwindCSS setup
- Fully responsive design
- HD images in /public/images
- Functional Booking Form (frontend-only)
